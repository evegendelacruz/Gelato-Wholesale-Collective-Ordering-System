🚀 GET STARTED IN 3 STEPS
═══════════════════════════════════════════════════════════════════════════

STEP 1: Setup Backend Service
───────────────────────────────────────────────────────────────────────────
   cd bartender-print-service
   npm install

   Then:
   1. Create "refs" folder
   2. Copy BarTender.Print.dll to refs/
   3. Copy .env.example to .env
   4. Edit .env with your paths


STEP 2: Configure Web App
───────────────────────────────────────────────────────────────────────────
   Edit .env.local and add:

   NEXT_PUBLIC_BARTENDER_API_URL=http://localhost:3001
   NEXT_PUBLIC_BARTENDER_TEMPLATE_PATH=C:\\...\\Sample.btw
   NEXT_PUBLIC_DEFAULT_PRINTER=Toshiba B-415


STEP 3: Start Printing!
───────────────────────────────────────────────────────────────────────────
   1. Double-click: bartender-print-service\START_SERVICE.bat
   2. Start web app: npm run dev
   3. Click "🖨️ BarTender SDK Print" button
   4. Watch labels print automatically! 🎉

🎯 HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════

   User → Web App → HTTP API → Print Service → BarTender → Printer

   1. User clicks "Print" button in web browser
   2. Web app sends label data to print service (HTTP POST)
   3. Print service calls BarTender SDK (Node.js → .NET)
   4. BarTender formats labels using BTW template
   5. BarTender sends to Toshiba B-415 printer
   6. Labels print automatically!

   Result: TRUE one-click automated printing! 🖨️

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Service won't start?
───────────────────────────────────────────────────────────────────────────
   → Run: npm install
   → Check BarTender Automation is installed
   → Try running as Administrator

Web app says "Service offline"?
───────────────────────────────────────────────────────────────────────────
   → Start service: START_SERVICE.bat
   → Check: http://localhost:3001/health
   → Verify .env.local settings

Prints blank labels?
───────────────────────────────────────────────────────────────────────────
   → Check template field names
   → Verify "Named Data Source" type
   → Test print in BarTender Designer

Template not found?
───────────────────────────────────────────────────────────────────────────
   → Check NEXT_PUBLIC_BARTENDER_TEMPLATE_PATH
   → Use full path with double backslashes
   → Verify file exists

Printer not found?
───────────────────────────────────────────────────────────────────────────
   → Check exact printer name (case-sensitive!)
   → Test print from Windows first

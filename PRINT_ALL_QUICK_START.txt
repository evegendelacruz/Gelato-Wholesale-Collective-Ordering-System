┌───────────────────────────────────────────────────────────────────────────┐
│                      📋 IMPORTANT SETTINGS                                │
└───────────────────────────────────────────────────────────────────────────┘

When printing from BarTender:

✅ Printer:       Toshiba B-415
✅ Paper Size:    90mm × 50mm
✅ Orientation:   Landscape
✅ Copies:        1  ⚠️ ALWAYS SET TO 1!
✅ Scaling:       Actual Size (100%)
✅ Quality:       Best/High

WHY Copies = 1?
   → PDF already has correct number of pages
   → If you ordered 10 labels, PDF has 10 pages
   → Setting Copies > 1 will print duplicates!

┌───────────────────────────────────────────────────────────────────────────┐
│                      ⚠️ IMPORTANT TIPS                                    │
└───────────────────────────────────────────────────────────────────────────┘

🔴 CRITICAL - Set Copies = 1
   Each PDF page = one label
   The PDF already has the correct number of pages
   Copies > 1 will create duplicates!

✅ Use "Actual Size" or "100%" Scaling
   Don't use "Fit to Page"
   Labels must be exactly 90mm × 50mm

✅ Check Print Preview First
   Verify label size and layout
   Make sure all pages are there

✅ Load Label Stock Before Printing
   Ensure printer has 90mm × 50mm labels
   Check label orientation


┌───────────────────────────────────────────────────────────────────────────┐
│                      🐛 TROUBLESHOOTING                                   │
└───────────────────────────────────────────────────────────────────────────┘

Problem: PDF won't open
Solution: Install BarTender or use Adobe Reader

Problem: Labels are wrong size
Solution: Check printer settings → 90mm × 50mm, Actual Size

Problem: Some labels missing
Solution: Check PDF page count (should match order quantity)

Problem: Labels printing twice
Solution: Set Copies = 1 (not 2!)

Problem: Can't find the button
Solution: Look for GREEN button "🎯 BarTender Print All"

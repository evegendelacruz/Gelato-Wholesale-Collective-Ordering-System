⚙️ IMPORTANT SETTINGS:
═══════════════════════════════════════════════════════════════════════════

When printing from BarTender:

   Printer:      Toshiba B-415
   Paper Size:   90mm × 50mm
   Orientation:  Landscape
   Copies:       1  ⚠️ ALWAYS SET TO 1!
   Scaling:      Actual Size (100%)

WHY Copies = 1?
   → PDF already has correct number of pages
   → Each page = one label
   → If order is 10 units, PDF has 10 pages
   → Setting Copies > 1 will print duplicates!

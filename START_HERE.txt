│                      ⚡ 30-SECOND QUICK TEST                              │
└───────────────────────────────────────────────────────────────────────────┘

Want to test if everything works? Follow these steps:

1. Open BarTender (install first if needed)

2. Load the template:
   → Open: public/assets/Sample.btw

3. Connect test data:
   → File → Database Connection Setup
   → Add Text File connection
   → Browse to: public/assets/bartender/SAMPLE_TEST_DATA.csv
   → Settings: ✓ First row = field names, Delimiter = Comma

4. Print Preview:
   → File → Print Preview
   → Should see 3 test labels with data

5. Print one label:
   → File → Print
   → Printer: Toshiba B-415
   → Copies: 1  (⚠️ ALWAYS SET TO 1!)
   → Click Print

✅ If the label prints correctly at 90mm × 50mm, you're all set!

┌───────────────────────────────────────────────────────────────────────────┐
│                     ⚠️ IMPORTANT NOTES                                    │
└───────────────────────────────────────────────────────────────────────────┘

🔴 CRITICAL - ALWAYS SET COPIES = 1
   The CSV already contains the correct number of labels!
   Each row in the CSV = one label to print
   Setting Copies > 1 will print duplicates

✅ Field Names Are Case-Sensitive
   Use exactly: CompanyName, ProductName, Allergens, etc.
   Not: companyname, product_name, ALLERGENS

✅ Label Size Must Be Correct
   Always: 90mm × 50mm (landscape)
   Printer: Toshiba B-415

✅ Fonts Must Be Installed
   Install from public/assets/:
   • ARIALN.TTF, ARIALNB.TTF
   • ARIAL.TTF, ARIALBD.TTF
